# Brainwave Tuner

A PICO-8 micro-game available to [play on itch.io](https://tinymicrobe.itch.io/brainwave).

## Usage

To start PICO-8 from this project, run:

```bash
./run.sh
```

## Export Game

Inside the PICO-8 cli, type:
```bash
# Be sure to save a cartridge image with F7 first!
./export.sh
```
